## Featured Project — Interactive Resume
 
An animated, storytelling take on the traditional resume — built to show, not just tell.
 
**[→ View Live Demo](https://tedi-lowney-interactive-resume.vercel.app/)** · **[→ View Source](https://github.com/Tedicode/claude-nextjs-resume)**
 
- Built with **Next.js 14 (App Router)**, **Framer Motion**, and **Tailwind CSS**
- Expand-in-place animations reveal visual storytelling behind each resume bullet
- Content layer separated into a typed `stories.ts` data module for easy iteration
- Deployed on Vercel with OpenGraph image generation via `next/og`
